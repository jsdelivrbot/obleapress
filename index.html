A simple and lightweight blank starter theme with sass, webpack, jQuery, AOS.JS and anime.js support to develop faster.

Forget about vendor prefixes.

Webpack help’s to not worry about to vendor prefixes for support in modern browsers.

Real-Time development.

Everything you do, will be real time in any browser with the same Wi-fi.

Sass Support.

Use DRY development with sass or scss.

Obleapress in a project.

You only need to type 2 commands in your terminal

1- npm run watch: for real time development

2- npm run production: for css and js production.

Don’t forget to install npm dependencies:

npm install (using npm and node)

yarn install (using yarn and node)
